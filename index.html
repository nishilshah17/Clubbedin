About

General

The purpose of this app is to allow ease of communication between leaders and members of clubs. This application allows one to create events and meetings for their club and send notifications to members about upcoming ones, as well as track member attendance.

Getting Started

You can login to this application using your Google or Facebook credentials. After, you will be able to add a club to Your Clubs or create a club. If you create a club, the club is assigned a specific id number, which other members can use to the join the club on this same application. After this step, you will be able to add events to the club and/or receive notifications from the leader of your club. Have fun and stay organized with Clubbed In!
